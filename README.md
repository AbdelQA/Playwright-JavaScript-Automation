# PlaywrightJS

(NOTE: Work In Progress)

## Installing:

` npm install `

## Running Tests (Headless mode):

`npx playwright test loginTests.spec.js`

## Running Tests (in UI mode):

`npx playwright test loginTests.spec.js --ui`
